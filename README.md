# CodingCamp-9Mar26-Kaneysha
Mini Project Batch 09-03-2026

Making a mini web with Kiro.
Main features: 
1. Greeting
   - Show current time and date
   - Show a greeting based on the time of day
2. Focus Timer
   - 25-minute timer
   - Start, stop, and reset buttons
3. To-Do List
   - Add tasks
   - Edit tasks
   - Mark tasks as done
   - Delete tasks
   - Save tasks using Local Storage
4. Quick Links
   - Buttons that open favorite websites
   - Links must be saved in Local Storage

Additional features:
1. Custom name in greeting
   - Input your own name to be displayed in greeting
2. Prevent duplicate tasks
   - To-do list will display "This task already exists" message when user inputs task that has already there
3. Sort tasks
   - Tasks are sorted based on the time and status. The older tasks are displayed on top, meanwhile the newer tasks are displayed below the older tasks. The tasks that had already been ticked are going to the bottom.
